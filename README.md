## Features

- **10 Iconic F&F Cars** dari Dodge Charger Dom sampai Lykan HyperSport
- **Specs tiap mobil** horsepower, akselerasi 0 - 60, top speed
- **Neon cursor custom** titik merah + ring oranye dengan efek glow
- **Speed line animations** di hero section
- **Auto-scrolling marquee** nama mobil ikonik
- **Hover effects** di setiap card dengan zoom gambar
- **Power bar animated** saat scroll masuk viewport
- **Responsive** mobile friendly
- **Dark theme** dengan palet warna khas F&F: merah, oranye, kuning nitrous

## Cara Jalankan Lokal

1. Clone repo ini:
   ```bash
   git clone https://github.com/agusadhitama/iron-garage.git
   ```

2. Masuk ke folder:
   ```bash
   cd iron-garage
   ```

3. Buka `index.html` di browser atau pakai **Live Server** di VSCode untuk auto-reload.

> Tidak perlu install apapun. Zero dependencies.
